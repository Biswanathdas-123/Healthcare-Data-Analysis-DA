# Healthcare-Data-Analysis-DA

Healthcare data analysis project using Python and Pandas to identify disease trends, health patterns, and risk factors.

The goal of this project is to analyze healthcare patient data and identify:

- Disease trends
- Risk factors
- Health patterns
- Patient statistics
